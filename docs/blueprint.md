# **App Name**: ContentQA

## Core Features:

- Dashboard: Overview of projects, quick access to start a new audit.
- Brand Document Upload: Allow users to upload brand documents to establish a brand voice.
- URL Input: Enable users to enter URLs or upload a list to audit their content.
- AI-Powered Audit: Utilize generative AI to assess website content, measure compliance, suggest improvements and use reasoning with uploaded brand guidelines as a tool.
- Results Table: Display audit results, including scores, flagged issues, and suggested rewrites, in an interactive table.
- Page Detail Modal: Provide a detailed report for each audited page, highlighting issues and improvements.
- Export Results: Allow users to export audit results in CSV/PDF format for sharing and further analysis.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and authority.
- Background color: Light gray (#F5F5F5) for a clean, professional feel.
- Accent color: Teal (#009688) to highlight key elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, objective look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, concise icons for navigation and key features.
- Clean and structured layout with a focus on readability and usability.
- Subtle animations for loading states and user interactions.
